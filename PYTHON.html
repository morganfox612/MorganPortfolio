Rock Paper Scissors.

This is a Rock Paper Scissors game I created in my Python class. This game allows you to enter rock, paper, or scissors and play against the cpu. If you win the game congratulates you, and if you lose the game allows you to try again.


I was able to complete this using multiple if statements and some else if statements. The process of creating this game allowed me to better understand working with and evaluating user input in Python.
