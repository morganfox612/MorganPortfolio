Pokemon Gym.

This was a group project given by NYCDA. The purpose of this project was to combine out coding and design skills and create a fucntioning pokemon gym. Our gym includes three trainers(one per person), three pokemon per trainer, and three sections(one for each trainer).


The main role I had in this group was to create the template for our website and implement it throug code. This group helped me become familiar with working with other coders using GitHub repositories and branches.
